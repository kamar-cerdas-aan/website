# Aplikasi Web Smart Room

## Prasyarat
- NodeJS

## Cara Pasang

- Kloning repo ini
- Instal package dengan perintah ```npm install```
- Jalankan server dengan perintah
```bash
npm run dev
```
- Buka [http://localhost:3000](http://localhost:3000) di peramban web dan mulai login/register untuk memulai
